# Mesto Production

Учебный проект Mesto: интерактивная страница с профилем пользователя и карточками мест, подключенная к серверу Яндекс Практикума.

## Технологии

- HTML
- CSS
- JavaScript
- Vite
- API
- GitHub Pages

## Функциональность

- загрузка профиля пользователя с сервера;
- загрузка карточек с сервера;
- редактирование имени и описания профиля;
- обновление аватара;
- добавление новых карточек;
- удаление только своих карточек;
- постановка и снятие лайков;
- отображение счетчика лайков;
- просмотр изображения карточки в модальном окне;
- валидация всех форм;
- закрытие модальных окон по кнопке, оверлею и Escape;
- статистика карточек по клику на логотип сайта.

## Команды

```bash
npm install
npm run dev
npm run build
npm run deploy
```
